**Screenshot App**

**ðŸŽ‰** This is a simple Python app that takes screenshots of your screen.

**How to use it**

1. Run the app.
2. Click the "Take Screenshot" button.
3. A screenshot of your screen will be saved to the `screenshots_data` directory.
4. The screenshot will also be displayed in a new window.

**Features**

* Easy to use
* Lightweight
* Cross-platform

**Dependencies**

* Python 3.6+
* PyAutoGUI
* Tkinter
